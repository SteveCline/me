How I would fix Healthcare

2/10/2019

Healthcare is a huge

  • starts with a blank white HTML page
  • follows web standards and semantics
  • vanilla HTML, CSS, and JS
  • classes and ids have meaning
  • CSS and JS are seperate files from HTML
  • made by one person or a small team
  • elegant efforts to keep code base minimal as possible

What Artisanal Web Design generally is NOT:

  • frameworks, templates, or CMS
  • a build step
  • needs transpiled or converted for a browser
  • use of libraries such as JQuery, Bootstrap, React, Angular
  • CSS mixed in JS
  • use of package managers
  • large teams

I am sure many web developers will identify with this term and I am sure some not so much. The web is a big place and there seem like 10,000 ways to make a webpage. Artisnal Web Design is just one way I want to define. Deciding which way is best, which skills should be valued most, and what is the best way for new folks to enter the industry is beyond my expertise. We all have differnet experiences and different reasons we got into the web. We all have opinons, I just hope we can be cival and polite. One thing I do feel confindent about, I think the future holds opportunity for everyone no matter the particular skillset they arm themself with.

Think of the artisan baker. They might not make as much money as a huge bread company, but they make a hand-crafted product they are proud of, even if it takes a bit longer and costs a bit more to produce. Maybe it is my hope that this term, Artisanal Web Design, gives value to those who feel undervalued and empowers them during this Great Divide.
